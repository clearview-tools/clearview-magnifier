## 下载与安装

### Chrome / Edge（推荐）

1. 打开 [Releases](https://github.com/clearview-tools/clearview-magnifier/releases) 下载 **`clearview-magnifier-v*.zip`**（不要下载 Source code）
2. 解压到本地文件夹；选中的文件夹里必须**直接能看到 `manifest.json`**
3. 打开 `chrome://extensions/`（Edge 为 `edge://extensions/`）→ 开启 **开发者模式**
4. **加载已解压的扩展程序** → 选择上述文件夹
5. 在普通 `https://` 网页上 **Alt + 右键** 启动放大镜

更新：下载新 ZIP 覆盖原目录，在扩展管理页点击 **重新加载**，然后刷新网页（F5）。

> 扩展无法在浏览器内置页面（如 `chrome://`）上运行。

### 360 浏览器（安全 / 极速）

360 浏览器可通过开发者模式加载本扩展。不同版本安全策略不同，扩展面板内提供**浏览器兼容补丁**，会自动检测并推荐：

| 浏览器 | 扩展管理页 | 推荐补丁 |
|--------|------------|----------|
| **360 极速浏览器** | `chrome://extensions/` | 360 极速补丁 |
| **360 安全浏览器** | `se://extensions/` | 360 安全补丁 |

**安装步骤**

1. 下载并解压 [Release ZIP](https://github.com/clearview-tools/clearview-magnifier/releases)（文件夹内需直接含 `manifest.json`）
2. 确认浏览器为**极速 / Chromium 模式**（地址栏旁闪电图标），不要用兼容 / IE 模式
3. 打开上表对应扩展管理页 → 开启**开发者模式** → **加载已解压的扩展程序**
4. 点击 ClearView 图标，在 **浏览器兼容** 区域查看检测结果，保持「自动」或点击**应用推荐补丁**
5. 打开 `https://` 网页，按 **F5** 后使用 **Alt + 右键** 测试

**仍无法加载？**

- 提示清单文件缺失 → 选错了文件夹，需进入能直接看到 `manifest.json` 的目录
- 截图 / 翻译失败 → 在面板切换对应 360 补丁，或改用 Chrome / Edge

## 功能特性

- **屏幕放大镜** — 圆形 / 圆角矩形透镜，跟随鼠标，可调放大倍数与透镜大小
- **实时翻译（Pro）** — 放大镜指向位置的文字自动翻译
- **阅读辅助** — 阅读辅助线、十字准星、高对比度 / 反色模式
- **实用工具** — 链接预览、颜色拾取、双击复制原文 + 译文
- **多种启动方式** — 鼠标组合键、快捷键、设置面板

## 快速上手

1. 在任意网页上按住 **Alt + 右键** 启动放大镜
2. 移动鼠标，透镜跟随光标
3. **滚轮** 调整放大倍数，**Shift + 滚轮** 调整透镜大小
4. 按 **Esc** 退出

## 设置

点击扩展图标打开面板，可配置：

- **翻译语言** — 源语言（支持自动检测）与目标语言
- **放大镜** — 默认放大倍数、透镜大小、透镜形状
- **松开后关闭** — 勾选后，松开 Alt+右键 即关闭（默认保持）
- **辅助功能** — 阅读线、十字准星、链接预览、颜色拾取、操作提示条等
- **浏览器兼容** — 360 等环境下的推荐补丁

Pro 用户可开启设置云同步（同一浏览器账号下多端共享）。会话中的滚轮 / 快捷键调节为临时值，下次启动仍使用面板默认值。

## 隐私

- 翻译仅将放大镜区域内的文字片段发送至第三方公共翻译服务
- 截图仅在本地处理，不上传
- 设置保存在浏览器本地或同步存储中

详见 [隐私政策](https://clearview-tools.github.io/clearview-magnifier/privacy.html)。

## 常见问题

**透镜不跟随 / 截图失败？**
- 请在普通网页使用，不要在 `chrome://` 等内置页面上使用
- 在扩展管理页重新加载扩展后，刷新当前网页（F5）

**翻译不可用？**
- 需激活 Pro，并在面板中勾选「启用实时翻译」
- 公共翻译服务有频率限制，稍后重试即可
- 360 安全浏览器请在面板启用「360 安全浏览器补丁」

**360 浏览器异常？**
- 扩展面板 → **浏览器兼容** → 选择推荐补丁或保持「自动」
- 必须使用极速 / Chromium 模式

**下载后无法加载？**
- 请下载 Release 中的 **`.zip` 安装包**，不要下载 Source code
- 加载时选中解压后**直接包含 `manifest.json` 的文件夹**
